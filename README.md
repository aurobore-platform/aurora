# Aurobore

> Открытая платформа для создания **WebView-приложений под ОС Аврора** на стандартном вебе
> (HTML/CSS/JS/TS) с доступом к нативным возможностям через типизированный мост и систему плагинов.
> Аналог Cordova / Capacitor / Tauri, спроектированный с учётом особенностей Аврора (Qt/QML + C++).

**Статус: этап проектирования.** Код пока не пишется — сначала фундамент-архитектура.
Вся документация находится в каталоге [`docs/`](docs/README.md).

## Требования к окружению разработчика

> **Сейчас (этап Design) кода нет** — для работы с документацией достаточно **редактора + Git**.
> Список ниже понадобится с началом реализации (MVP) и для сборки/запуска под ОС Аврора.
> Подробности: [docs/aurora/sdk-overview.md](docs/aurora/sdk-overview.md),
> [docs/aurora/build-and-packaging.md](docs/aurora/build-and-packaging.md).

### Общее (Windows и Linux)

| Что                                                        | Зачем                                                                                               | Версия                                                                             |
| ---------------------------------------------------------- | --------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| **Git**                                                    | контроль версий                                                                                     | актуальная                                                                         |
| **Node.js**                                                | рантайм инструментария `@aurobore/*` (CLI, SDK, кодоген) — [ADR-005](docs/adr/ADR-005-cli-stack.md) | LTS, рекоменд. **20+** _(точный минимум фиксируется в инфраструктурном чек-листе)_ |
| **pnpm**                                                   | менеджер пакетов monorepo (workspaces); удобно через `corepack enable`                              | актуальная                                                                         |
| **Аврора SDK** (вариант **mb2 / Build Engine**, на Docker) | штатная сборка приложения и RPM, эмулятор (QEMU) — [ADR-007](docs/adr/ADR-007-packaging-build.md)   | актуальная (5.2.x)                                                                 |
| **Сертификаты подписи RPM**                                | установка пакета на устройство/эмулятор обязательна с подписью                                      | —                                                                                  |

> SDK 5.2.0+ рекомендуется запускать **на хосте**, а не внутри виртуальной машины.

### Windows

| Что                                               | Зачем                                                                                  |
| ------------------------------------------------- | -------------------------------------------------------------------------------------- |
| **Windows 10 build ≥ 17063**                      | требование Аврора SDK (включает архиватор `tar`)                                       |
| **Git Bash**                                      | команды `mb2`/`sb2` требуют Unix-совместимый терминал (PowerShell для них не подходит) |
| **Режим разработчика** (или политика на symlinks) | избежать ошибок из-за символических ссылок при сборке                                  |
| `ssh-keygen` (идёт с Git)                         | запрашивается мастером установки SDK                                                   |

### Linux

| Что                         | Зачем                                                    |
| --------------------------- | -------------------------------------------------------- |
| **Docker**                  | виртуализация среды сборки (Build Engine) и эмулятора    |
| Стандартный терминал        | команды `mb2`/`sb2` работают нативно (Git Bash не нужен) |
| Поддержка KVM/виртуализации | ускоряет эмулятор (QEMU)                                 |

### По приоритету

1. **Сейчас (Design):** редактор + Git.
2. **Старт реализации (MVP):** + Node.js (LTS) + pnpm.
3. **Сборка/запуск под Аврору:** + Аврора SDK (mb2) + эмулятор + сертификаты подписи (+ на Windows: Git Bash и режим разработчика).
