# Share Demo

Демо плагина `@aurobore/share` (P5): системный share sheet для текста, URL и файла из app-data.

```powershell
pnpm build:web
aurobore build
aurobore run
```

На устройстве/эмуляторе: кнопки открывают системный share UI. **Share file** использует `Echo.getSampleResource()` → resource URL в app-data.
