# Aurobore > Открытая платформа для создания **WebView-приложений под ОС Аврора** на стандартном вебе > (HTML/CSS/JS/TS) с доступом к нативным возможностям через типизированный мост и систему плагинов. > Аналог Cordova / Capacitor / Tauri, спроектированный с учётом особенностей Аврора (Qt/QML + C++). **Статус: MVP (M0–M5).** Платформа включает runtime, мост, плагины, CLI, SDK и шаблоны. Документация — в [`docs/`](docs/README.md).
