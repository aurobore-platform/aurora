# @aurobore/{{name}}

{{display}} plugin — scaffold generated by `aurobore plugin create`.

## Develop

1. Edit `plugin.manifest` (source of truth), then run `aurobore generate` in your app project.
2. Implement native logic in `native/{{DISPLAY}}Plugin.cpp`.
3. Rebuild the app: `aurobore build` / `aurobore run`.

## Monorepo (Aurobore platform)

If this plugin lives in the Aurobore monorepo, `plugin create` also updates `PLUGIN_NAMES` and `runtime/container/CMakeLists.txt`. Then:

```bash
pnpm codegen:plugins
pnpm container:build
```

## Standalone app project

Add to `package.json`:

```json
"dependencies": {
  "@aurobore/{{name}}": "file:./plugins/{{name}}"
}
```

Then `aurobore plugin add {{name}}` and `aurobore build`.

If native staging cannot find the plugin, set `AUROBORE_PLUGINS_ROOT` to the directory containing `{{name}}/`.
