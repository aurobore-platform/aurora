# tests/ — сквозные и совместимостные тесты

- `e2e/` — сквозные сценарии; оркестратор `pnpm e2e:verify` (см. [e2e/README.md](e2e/README.md)).
- `conformance/` — тесты соответствия плагинов манифесту и поведению моста (FR-T1).
- `fixtures/` — тестовые данные/проекты.

Модульные тесты живут рядом с кодом в каждом пакете (`*.test.ts`).
